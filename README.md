# React-chat

A real time chat made with socket.io
The front is in React and the back in NodeJS    

Test the app at : https://realtime-reactchat.netlify.com/

## Installation

1. Install dependencies
    ```javascript
    npm install
    ```

2. Run back and front
    ```javascript
    npm start
    ```
